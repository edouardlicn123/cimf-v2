在 Django shell 中执行：
```python
from core.models import Taxonomy, TaxonomyItem

# 居民信息模块应创建的词汇表
expected_taxonomies = [
    ('resident_relation', '与户主关系', 6),
    ('resident_type', '人员类型', 2),
    ('grid', '所属网格', 5),
    ('key_category', '重点类别', 8),
    ('nation', '民族', 57),
    ('political_status', '政治面貌', 4),
    ('marital_status', '婚姻状况', 4),
    ('education', '文化程度', 7),
    ('health_status', '健康状况', 5),
    ('other_id_type', '其他证件类型', 4),
]

print("验证居民信息模块词汇表：")
print("=" * 60)
success_count = 0
failed_count = 0

for slug, expected_name, expected_count in expected_taxonomies:
    tax = Taxonomy.objects.filter(slug=slug).first()
    if not tax:
        print(f"✗ {slug}: 不存在！")
        failed_count += 1
        continue
    
    actual_count = tax.items.count()
    if tax.name != expected_name:
        print(f"⚠ {slug}: 名称不匹配（期望: {expected_name}，实际: {tax.name}）")
    
    if actual_count != expected_count:
        print(f"✗ {slug}: 词汇项数量不匹配（期望: {expected_count}，实际: {actual_count}）")
        failed_count += 1
    else:
        print(f"✓ {slug}: {tax.name} ({actual_count} 项)")
        success_count += 1

print("=" * 60)
print(f"结果：成功 {success_count} 个，失败 {failed_count} 个")

if failed_count == 0:
    print("✓ 所有词汇表验证通过！")
else:
    print("✗ 存在验证失败的词汇表")
```

**预期结果**: 10个词汇表全部创建成功，共102个词汇项
