# 模块化设计方案（扩展版）

## 问题背景

当前系统存在大量硬编码模块引用，每新增一个模块都需要修改多个核心文件。

### 硬编码位置汇总

| 文件 | 行号 | 硬编码类型 | 说明 |
|------|------|-----------|------|
| `core/importexport/services.py` | 51-53, 416-424, 603-611 | 字典键、导入语句 | 导出字段映射、服务类映射 |
| `core/importexport/model_registry.py` | 27-38 | 条件判断、导入语句 | 模型类映射 |
| `core/node/services.py` | 36, 52, 245 | slug字符串 | 默认节点类型 |

---

## 设计目标

1. **模块自包含**：所有模块相关配置定义在模块内部
2. **动态加载**：系统运行时动态读取模块配置
3. **易于扩展**：新增模块无需修改核心代码
4. **向后兼容**：保持现有功能正常工作

### 2. 修改核心文件，动态读取配置

#### 2.1 修改 ModelRegistry 动态获取模型

**文件**：`core/importexport/model_registry.py`

```python
@classmethod
def get_model(cls, slug: str):
    """动态获取模型类"""
    if slug in cls._registry and cls._registry[slug] is not None:
        return cls._registry[slug]
    
    # 动态导入模块模型
    try:
        from importlib import import_module
        mod = import_module(f'modules.{slug}.models')
        
        # 查找 Fields 结尾的模型类
        for attr_name in dir(mod):
            attr = getattr(mod, attr_name)
            if (attr_name.endswith('Fields') and 
                hasattr(attr, '_meta') and 
                hasattr(attr, 'node')):
                cls._registry[slug] = attr
                return attr
    except (ImportError, ModuleNotFoundError):
        pass
    
    return None
```

#### 2.2 修改 ExportService 动态读取配置

**文件**：`core/importexport/services.py`

```python
@classmethod
def get_exportable_fields(cls, node_type_slug: str) -> List[Dict]:
    """动态获取可导出的字段列表"""
    from importlib import import_module
    
    try:
        mod = import_module(f'modules.{node_type_slug}.module')
        if hasattr(mod, 'MODULE_INFO'):
            return mod.MODULE_INFO.get('export_fields', [])
    except (ImportError, ModuleNotFoundError):
        pass
    
    return []

@classmethod
def _get_service_class(cls, node_type_slug: str):
    """动态获取服务类"""
    try:
        from importlib import import_module
        mod = import_module(f'modules.{node_type_slug}.services')
        
        # 查找 Service 结尾的类
        for attr_name in dir(mod):
            attr = getattr(mod, attr_name)
            if (attr_name.endswith('Service') and 
                hasattr(attr, 'get_list')):
                return attr
    except (ImportError, ModuleNotFoundError):
        pass
    
    return None
```

    from core.node.services import ModuleService
    modules = ModuleService.get_all_installed()

#### 2.4 修改 core/views.py 动态获取统计

**文件**：`core/views.py`

```python
# 动态获取支持统计的模块
def get_modules_with_stats():
    """获取支持统计的模块"""
    from core.node.services import ModuleService
    from importlib import import_module
    
    modules_with_stats = []
    installed_modules = ModuleService.get_all_installed()
    
    for module in installed_modules:
        if not module.is_installed or not module.is_active:
            continue
        
        try:
            mod = import_module(f'modules.{module.module_id}.module')
            if hasattr(mod, 'MODULE_INFO'):
                module_info = mod.MODULE_INFO
                if module_info.get('dashboard_stats', False):
                    modules_with_stats.append(module.module_id)
        except (ImportError, ModuleNotFoundError):
            pass
    
    return modules_with_stats
```
