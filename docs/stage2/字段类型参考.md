# 字段类型快速参考

> 开发 Node 模块时，字段必须从以下类型中选取。可通过 `/nodes/field-types` 查看最新列表。

## 文本类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `string` | 单行文本 | input | value |
| `string_long` | 多行纯文本 | textarea | value |
| `text` | 带格式文本 | input | value, format |
| `text_long` | 带格式长文本 | textarea | value, format |
| `text_with_summary` | 含摘要文本 | textarea | value, summary, format |

## 数值类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `boolean` | 布尔值 | checkbox | value |
| `integer` | 整数 | input | value |
| `decimal` | 精确小数 | input | value |
| `float` | 浮点数 | input | value |

## 联系方式类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `email` | 邮箱 | input | value |
| `telephone` | 电话 | input | value |
| `address` | 标准地址 | address_input | province, city, district, street, house_number, grid_id |

## 文件媒体类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `file` | 文件 | file_input | target_id, display, description |
| `image` | 图片 | image_input | target_id, alt, title, width, height |
| `link` | 链接 | input | uri, title, options |

## 日期时间类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `datetime` | 日期时间 | datetime_input | value |
| `timestamp` | 时间戳 | datetime_input | value |

## 关联类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `entity_reference` | 关联引用 | select | target_id, target_type |

## 特殊类

| 类型标识 | 显示名称 | 控件 | 属性 |
|----------|----------|------|------|
| `color` | 颜色选择器 | color_picker | color_code, opacity |
| `identity` | 证件识别码 | input | id_number, id_type, is_verified |
| `masked` | 隐私脱敏字段 | input | raw_value, display_value, permission_level |
| `biometric` | 生物特征引用 | input | feature_vector, type, version |

## 延后类（暂未完整实现）

| 类型标识 | 显示名称 | 说明 |
|----------|----------|------|
| `geolocation` | 地理位置 | 需引入地图库 |
| `ai_tags` | 智能标签 | 需接入 AI 服务 |
| `gis` | 地理围栏 | 需地图功能完善 |

## 使用示例

```python
# 在 Node 定义中使用字段
class ProjectNode:
    fields = {
        'name': {'type': 'string', 'label': '项目名称', 'required': True},
        'description': {'type': 'text', 'label': '项目描述'},
        'budget': {'type': 'decimal', 'label': '预算'},
        'start_date': {'type': 'datetime', 'label': '开始日期'},
        'cover_image': {'type': 'image', 'label': '封面图片'},
    }
```
