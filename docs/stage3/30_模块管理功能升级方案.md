# 模块管理功能升级方案

## 概述

将模块概念升级为更通用的功能组件，可表示内容类型或系统子功能。并优化模块管理页面的导航、权限控制和启用/禁用逻辑。

## 概念定义

### 模块 (Module)
- 模块可以是某种**内容类型**（如客户、项目等业务数据）
- 模块也可以是系统的**子功能组件**（如导入导出、报表等系统功能）
- 模块具有独立的代码结构（module.py、views.py、models.py 等）
- 模块需要安装和启用才能生效

### 模块与节点类型的关系
```
模块 (NodeModule)
    ↓ 安装
节点类型 (NodeType) - 对应数据库表和业务数据
    ↓ 启用
节点数据可见 - 用户可访问该类型的节点数据
```

### 统一启用/禁用规则
- **启用模块** → 同步启用对应的节点类型
- **禁用模块** → 同步禁用对应的节点类型
- 两者状态始终保持一致

### 2. 权限控制

#### 页面权限
- **查看权限**: 仅 `is_admin=True` 的用户可访问
- **操作权限**: 所有模块管理操作（扫描、安装、启用、禁用、新建）都需要管理员权限
- 使用 `PermissionService.can_access_admin()` 进行权限检查

#### 现有视图权限（无需修改）
以下视图已使用 `can_access_admin` 装饰器：
- `node_modules` - 模块管理页
- `module_scan` - 扫描模块
- `module_install` - 安装模块
- `module_enable` - 启用模块
- `module_disable` - 禁用模块
- `module_create` - 新建模块
- `module_create_action` - 创建模块动作

### 3. 启用/禁用同步逻辑

#### 修改 services.py

**enable_module() 函数增强：**
```python
@staticmethod
def enable_module(module_id: str) -> Optional[NodeModule]:
    """启用模块并同步节点类型"""
    try:
        module = NodeModule.objects.get(module_id=module_id)
        if module.is_installed:
            module.is_active = True
            module.activated_at = timezone.now()
            module.save(update_fields=['is_active', 'activated_at'])
            
            # 同步更新节点类型
            node_type = NodeType.objects.filter(slug=module.module_id).first()
            if node_type:
                node_type.is_active = True
                node_type.save(update_fields=['is_active'])
            
            return module
    except NodeModule.DoesNotExist:
        pass
    return None
```

**disable_module() 函数增强：**
```python
@staticmethod
def disable_module(module_id: str) -> Optional[NodeModule]:
    """禁用模块并同步节点类型"""
    try:
        module = NodeModule.objects.get(module_id=module_id)
        module.is_active = False
        module.save(update_fields=['is_active'])
        
        # 同步更新节点类型
        node_type = NodeType.objects.filter(slug=module.module_id).first()
        if node_type:
            node_type.is_active = False
            node_type.save(update_fields=['is_active'])
        
        return module
    except NodeModule.DoesNotExist:
        pass
    return None
```
