## 实施计划

### 步骤1：创建 TaxonomyService 服务层

**文件**: `core/services/taxonomy_service.py`

从旧版移植 `TaxonomyService` 类，包含：
- `get_all_taxonomies()`
- `get_taxonomy_by_id()`
- `get_taxonomy_by_slug()`
- `create_taxonomy()`
- `update_taxonomy()`
- `delete_taxonomy()`
- `get_items()` / `create_item()` / `update_item()` / `delete_item()`
- `reorder_items()`

### 步骤2：添加默认词汇表数据

**文件**: `core/services/taxonomy_service.py`

添加 `DEFAULT_TAXONOMIES` 常量，包含全部 37 个词汇表的定义。

### 步骤3：添加初始化方法

**文件**: `core/services/taxonomy_service.py`

```python
@staticmethod
def init_default_taxonomies() -> None:
    """初始化预置分类数据"""
    for tax_data in DEFAULT_TAXONOMIES:
        # 检查是否已存在
        existing = Taxonomy.objects.filter(slug=tax_data['slug']).first()
        if existing:
            continue
        # 创建词汇表和词汇项
        ...
```

### 步骤4：更新视图

**文件**: `core/views.py`

需要添加词汇项的 CRUD 视图：
- `taxonomy_item_create` - 创建词汇项
- `taxonomy_item_update` - 更新词汇项
- `taxonomy_item_delete` - 删除词汇项
- `taxonomy_item_reorder` - 排序词汇项

### 步骤5：更新模板

#### 5.1 修改 view.html - 添加词汇项管理功能

- 添加词汇项表单（名称、描述）
- 添加词汇项列表（支持编辑、删除）
- 添加 AJAX 交互

#### 5.2 优化 index.html

- 添加词汇项数量显示
- 添加返回按钮

### 步骤6：更新 URL 路由

**文件**: `core/urls.py`

添加词汇项的路由：
```python
path('taxonomy/<int:taxonomy_id>/items', views.taxonomy_items, name='taxonomy_items'),
path('taxonomy/<int:taxonomy_id>/item/create', views.taxonomy_item_create, name='taxonomy_item_create'),
path('taxonomy/<int:taxonomy_id>/item/<int:item_id>', views.taxonomy_item_detail, name='taxonomy_item_detail'),
```

### 步骤7：更新初始化脚本

**文件**: `init_db.py`

在初始化数据时调用：
```python
from core.services.taxonomy_service import TaxonomyService
TaxonomyService.init_default_taxonomies()
```
