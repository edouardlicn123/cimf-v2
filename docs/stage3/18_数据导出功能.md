# 数据导出功能设计方案

> 状态：已实现
> 版本：1.3

---

## 一、功能概述

数据导出功能允许用户将已启用的节点模块（如客户信息（海外）、客户信息（国内））的数据导出为 CSV 或 XLSX 格式的文件，支持按字段筛选数据。

### 1.1 核心特点

- **通用性**：适用于所有已启用的节点模块
- **字段可选**：用户可选择要导出的字段（默认全选）
- **数据筛选**：支持多条件组合筛选（AND 关系）
- **格式多样**：支持 CSV 和 XLSX 格式
- **FK 显示友好**：外键字段显示名称而非 ID
- **JSON 字段展开**：如省市区 JSON 字段展开为文本

### 1.2 适用场景

- 数据备份
- 数据分析
- 与其他系统交换数据
- 报表生成

## 二、页面流程

### 2.1 完整流程

```
导出页（卡片列表）
    │
    ├── 字段选择页（默认全选，打勾选择 + 筛选条件）
    │       │
    │       ├── [上一步] → 返回导出页
    │       └── [下一步] → 保存 session → 进入确认页
    │
    ├── 确认页（显示摘要信息、筛选条件、数据预览）
    │       │
    │       ├── [上一步] → 返回字段选择页
    │       └── [开始导出] → 进入导出中页
    │
    ├── 导出中页（显示进度状态）
    │       │
    │       └── 完成 → 自动下载文件
    │
    └── 完成
```

### 2.2 URL 设计

```
/nodes/export/                                      # 导出页（卡片列表）
/nodes/export/<node_type_slug>/                     # 字段选择页
/nodes/export/<node_type_slug>/confirm/            # 确认页
/nodes/export/<node_type_slug>/exporting/          # 导出中页
/nodes/export/<node_type_slug>/do/                 # 执行导出（下载文件）
```

### 2.3 Session 使用

```python
# 导出流程中的 session 使用
# 1. 字段选择页 POST -> 保存到 session
request.session['export_selected_fields'] = ['customer_name', 'customer_code', ...]
request.session['export_format'] = 'xlsx'
request.session['export_filters'] = [
    {'field': 'customer_name', 'value': '测试'},
    {'field': 'region', 'value': '{"province":"广东省","city":"深圳市","district":""}'},
]

# 2. 确认页 GET -> 从 session 读取

# 3. 导出中页 GET -> 从 session 读取

# 4. 执行导出 GET -> 从 session 读取，执行后清除
del request.session['export_selected_fields']
del request.session['export_format']
del request.session['export_filters']
```

```python
    'entity_reference', 'datetime', 'date',

```

### 4.2 可筛选字段类型

仅字符串类型字段支持模糊筛选：

```python
FILTERABLE_FIELD_TYPES = [
    'string', 'string_long', 'text', 'email', 'telephone',
]

FILTERABLE_FK_FIELDS = [
    'customer_type', 'country', 'enterprise_type', 'customer_level',
]
```

### 4.3 导出字段映射

| 字段类型 | 导出表现 | 示例 |
|----------|----------|------|
| string | 直接显示 | "张三" |
| email | 直接显示 | "test@example.com" |
| telephone | 直接显示 | "13800138000" |
| FK (entity_reference) | 显示名称 | FK ID=5 → "代理商" |
| JSON (region) | 展开为文本 | "广东省 深圳市 南山区" |
| datetime | 直接显示 | "2024-01-15 10:30:00" |
| decimal | 数字格式 | "10000.00" |

### 4.4 筛选条件格式

```python
# 普通字段筛选
{'field': 'customer_name', 'value': '测试'}

# 省市区筛选（JSON 字符串）
{'field': 'region', 'value': '{"province":"广东省","city":"深圳市","district":""}'}
```

## 五、服务层设计

### 5.1 ExportService

```python
class ExportService:
    """数据导出服务"""
    
    FORMAT_CSV = 'csv'
    FORMAT_XLSX = 'xlsx'
    
    FK_FIELD_NAMES = ['customer_type', 'country', 'enterprise_type', 'customer_level']
    
    FILTERABLE_FIELD_TYPES = ['string', 'string_long', 'text', 'email', 'telephone']
    
    FILTERABLE_FK_FIELDS = ['customer_type', 'country', 'enterprise_type', 'customer_level']
    
    @classmethod
    def get_exportable_fields(cls, node_type_slug: str) -> List[Dict]:
        """获取可导出的字段列表"""
    
    @classmethod
    def get_filterable_fields(cls, node_type_slug: str) -> List[Dict]:
        """获取可筛选的字段列表"""
    
    @classmethod
    def has_region_field(cls, node_type_slug: str) -> bool:
        """检查节点类型是否有省市区字段"""
    
    @classmethod
    def get_fields_info(cls, node_type_slug: str, field_names: List[str]) -> List[Dict]:
        """获取选中字段的详细信息"""
    
    @classmethod
    def get_preview(cls, node_type_slug: str, field_names: List[str],
                    filters: List[Dict] = None, limit: int = 5) -> List[Dict]:
        """获取数据预览"""
    
    @classmethod
    def get_record_count(cls, node_type_slug: str, filters: List[Dict] = None) -> int:
        """获取记录总数"""
    
    @classmethod
    def export(cls, node_type_slug: str, field_names: List[str],
               export_format: str = 'csv', filters: List[Dict] = None) -> HttpResponse:
        """执行导出"""
    
    @classmethod
    def generate_filename(cls, node_type_slug: str, export_format: str) -> str:
        """生成导出文件名"""
    
    @classmethod
    def _get_filtered_queryset(cls, node_type_slug: str,
                               filters: List[Dict] = None) -> QuerySet:
        """获取应用筛选条件后的 QuerySet"""
    
    @classmethod
    def _convert_to_row(cls, item, field_names: List[str],
                         node_type_slug: str = None) -> Dict:
        """将数据对象转换为导出行"""
    
    @classmethod
    def _get_field_value(cls, item, field_name: str) -> Any:
        """获取字段值"""
    
    @classmethod
    def _resolve_fk_field(cls, item, field_name: str) -> str:
        """解析 FK 字段，返回名称"""
    
    @classmethod
    def _resolve_region_field(cls, item) -> str:
        """解析省市区 JSON 字段"""
```

### 8.1 全选/取消全选

```javascript
function toggleAllFields() {
    const checkboxes = document.querySelectorAll('.field-checkbox');
    const allChecked = Array.from(checkboxes).every(cb => cb.checked);
    checkboxes.forEach(cb => {
        cb.checked = !allChecked;
    });
}
```

### 8.2 筛选器互斥逻辑

```javascript
function onFilterFieldChange(index) {
    // 启用/禁用对应的输入框
    const input = document.querySelector(`.filter-group[data-index="${index}"] .filter-value`);
    input.disabled = !select.value;
    if (!select.value) input.value = '';
    
    // 更新其他下拉的选项（排除已选字段）
    updateFilterOptions();
}

function updateFilterOptions() {
    const selectedFields = [];
    document.querySelectorAll('.filter-field').forEach(sel => {
        if (sel.value) selectedFields.push(sel.value);
    });
    
    document.querySelectorAll('.filter-option').forEach(opt => {
        const fieldName = opt.value;
        if (selectedFields.includes(fieldName)) {
            const parentSelect = opt.closest('.filter-field');
            // 仅隐藏当前未选中该字段的其他下拉选项
            opt.style.display = parentSelect.value === fieldName ? '' : 'none';
        } else {
            opt.style.display = '';
        }
    });
}
```

### 8.3 字段提交处理

```javascript
document.getElementById('exportForm').addEventListener('submit', function(e) {
    // 移除 checkbox 原有的 name 属性（避免只取第一个值）
    // 通过 hidden input 传递逗号分隔的字段列表
    const hiddenInput = document.createElement('input');
    hiddenInput.type = 'hidden';
    hiddenInput.name = 'fields';
    hiddenInput.value = Array.from(document.querySelectorAll('.field-checkbox:checked'))
        .map(cb => cb.value).join(',');
    this.appendChild(hiddenInput);
});
```

## 十、注意事项

### 10.1 安全性

- 使用白名单机制，只允许导出指定类型的字段
- 不导出敏感字段（如密码相关）
- `@login_required` + 权限检查保护所有视图
- CSRF 保护（Django Jinja2 全局变量 `{{ csrf_token }}`）

### 10.2 性能

- 大数据量时使用分批查询（`queryset[:limit]` 限制预览数量）
- 导出时全量遍历，但 Django QuerySet 惰性加载

### 10.3 字段筛选限制

- **不支持筛选**：数字类型（integer/decimal/float）、布尔值、日期时间
- **支持筛选**：字符串类型（string/text/email/telephone）、FK 字段、省市区
- **原因**：`__icontains` 仅对字符串字段有语义意义

### 10.4 用户体验

- 字段默认全选，减少操作步骤
- 筛选条件下拉互斥，避免重复条件
- 确认页展示筛选条件和预览数据，用户可返回修改
- 导出中页自动跳转，无需用户操作
