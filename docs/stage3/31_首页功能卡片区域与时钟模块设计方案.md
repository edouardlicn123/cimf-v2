# 首页功能卡片区域与时钟模块设计方案

## 一、需求概述

### 1.1 功能卡片区域

在系统首页（仪表盘）添加一个 3×2 布局的功能卡片区域，共 6 个位置（编号 1-6），用于放置各种功能模块提供的卡片组件。

### 1.2 时钟模块

新建的时钟模块（clock）提供"时钟卡片"，挂载到功能卡片区域的指定位置。时钟卡片显示当前日期和时间，通过后台 API 获取实时数据。

### 1.3 拖拽功能

时钟卡片可在 6 个位置之间自由拖拽，拖拽完成后自动保存位置到用户设置。

### 1.4 模块启用/禁用

如果 clock 模块被禁用，首页不显示时钟卡片。

## 四、API 设计

### 4.1 获取当前时间（时钟模块）

**请求**

```
GET /modules/clock/api/time/
```

**响应**

```json
{
    "success": true,
    "data": {
        "datetime": "2026-03-27 19:30:45",
        "timestamp": 1772059845,
        "date": "2026-03-27",
        "time": "19:30:45",
        "weekday": "星期五",
        "weekday_en": "Friday",
        "month": "三月",
        "month_en": "March"
    }
}
```

### 4.2 保存功能卡片位置

**请求**

```
POST /api/user/dashboard/cards/
Content-Type: application/json

{
    "positions": {
        "1": {"module": "clock", "size": "medium"},
        "2": {"module": null, "size": "medium"},
        ...
    }
}
```

**响应**

```json
{
    "success": true,
    "message": "布局已保存"
}
```

**请求**


```json

            "1": {"module": "clock", "size": "medium"},
```

### 5.3 拖拽交互逻辑

```javascript
// 拖拽开始
function onDragStart(e) {
    e.dataTransfer.setData('text/plain', e.target.dataset.position);
    e.target.classList.add('dragging');
}

// 拖拽结束
function onDragEnd(e) {
    e.target.classList.remove('dragging');
}

// 放置
function onDrop(e) {
    const fromPosition = e.dataTransfer.getData('text/plain');
    const toPosition = e.target.closest('.card-slot').dataset.position;
    
    // 交换位置
    swapCards(fromPosition, toPosition);
    
    // 保存到服务器
    saveCardPositions();
}
```

### 5.4 时钟更新逻辑

```javascript
// 获取时间
async function updateClock() {
    const response = await fetch('/modules/clock/api/time/');
    const data = await response.json();
    
    if (data.success) {
        document.querySelector('.clock-date').textContent = data.data.date;
        document.querySelector('.clock-time').textContent = data.data.time;
        document.querySelector('.clock-weekday').textContent = data.data.weekday;
    }
}

// 每秒更新
setInterval(updateClock, 1000);
updateClock(); // 立即更新
```

```python
MODULE_INFO = {

        }
    ],

```

### 6.2 动态注册机制

首页加载时：
1. 查询所有已启用的模块
2. 检查模块是否有 `dashboard_cards` 配置
3. 获取用户保存的卡片布局设置
4. 渲染功能卡片区域

## 九、注意事项

1. **性能**：时钟更新使用间隔请求而非 WebSocket，减少服务器压力
2. **离线**：网络断开时显示最后更新时间
3. **权限**：卡片位置 API 需要登录验证
4. **兼容性**：拖拽功能使用原生 HTML5 API，兼容主流浏览器
