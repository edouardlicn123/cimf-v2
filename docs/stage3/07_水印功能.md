### 7.4 水印配置

#### 7.4.1 系统设置模型扩展

```python
# core/models.py 扩展

class SystemSetting(models.Model):
    # 现有字段...
    
    # 水印相关配置
    watermark_enabled = models.BooleanField(default=False)
    watermark_type = models.CharField(max_length=20, choices=[
        ('text', '文字水印'),
        ('image', '图片水印'),
    ])
    watermark_text = models.CharField(max_length=200, blank=True)
    watermark_image = models.ImageField(upload_to='watermarks/', blank=True)
    watermark_position = models.CharField(max_length=20, default='center')
    watermark_opacity = models.FloatField(default=0.3)
    watermark_font_size = models.IntegerField(default=48)
```
