# 水印功能测试清单

## 测试环境
- 操作系统：Linux/Windows/Mac
- 浏览器：Chrome/Edge/Firefox
- Python：3.12+
- 数据库：SQLite

## 测试前准备

1. 确保数据库已初始化并包含默认设置
2. 确保至少有2个测试用户（admin + 普通用户）
3. 清除浏览器缓存
